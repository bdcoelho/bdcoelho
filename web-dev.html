Home Search Organizer

An app for people looking to buy or rent an new home. House hunting can be a tedious and time consuming process. Many currently available applications cover constraints like price, suburb, number of bedrooms/baths/car spaces. However, travel time to frequently travelled locations for work, study or leisure is generally not provided in a convenient format. The page is being updated to allow multiple frequent travel locations and travel modes.

Weather Dashboard

The weather app is a handy tool for travellers wanting to factor in weather conditions while planning an itinerary. It's especially handy for travellers planning on multiple outdoor activities and moving between destinations. The app uses the Open Weather Map API to access current weather data and the Google maps API to reliably access the most relevant location.

Code Quiz

This project demonstrates the basic applications of javascript to make a dynamic and interactive interface while using local storage to personalize the application. Users can test their knowledge about javascript by answering a number of preselected questions and can store their scores locally.

Work Calendar

This project demonstrates the application of javascript to dynamically update a page and local storage. It's an easy way for an employee to schedule and keep track of their tasks for the day. The page is written so that the calendar can be dynamically generated. This will allow it to be easily modified to accommodate different work times and shift work if required.
